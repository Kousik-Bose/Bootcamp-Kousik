Visual Design - Key Notes

1. Recreating a Medium.com Layout

  • Designed a layout similar to Medium's article page with clean and minimalist styling.
  • Incorporated responsive design techniques to ensure a consistent look on all devices.
  • Structured the page content using semantic HTML for better accessibility and SEO.

2. Key Principles of Visual Hierarchy

  • Reading Patterns: Leveraged common reading patterns such as "F" and "Z" to guide the viewer's flow across content naturally.
  • Rule of Thirds: Divided the page into thirds and placed key design elements in focal areas to create visual balance.
  • Size and Scale: Used larger fonts and hero sections to highlight critical content and grab attention first.
  • Typography: Combined varied font sizes, weights, and styles to establish a clear text hierarchy.
  • Color and Contrast: Ensured text contrasts well with the background for readability while using color to emphasize key elements.
  • White Space: Added sufficient spacing between sections and elements to avoid clutter and improve clarity.
  • Proximity and Consistency: Grouped related components together and applied consistent styling for elements like buttons or cards.
  • Grid Layout: Organized the design using a CSS grid system to achieve alignment and responsive structure.
  • Responsive Design: Optimized the layout for smaller screens to ensure a user-friendly experience across devices.
