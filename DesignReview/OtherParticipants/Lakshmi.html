Design Review

Lakshmi's Webpage

Review Summary

Lakshmi's webpage demonstrates a good understanding of basic design principles but could benefit from improvements in structure and functionality.

EX-1: Fonts

Rating: 7/10

The font exercise was satisfactory, but it did not adhere to the CSS Zen Garden template guidelines. Better structuring of the information would improve the exercise.

EX-2: Colors

Rating: 7.5/10

The color exercise was well-executed, featuring good margins, padding, and font spacing. However, Level 2 functionality was incomplete, as the button did not work.

EX-3: Icons

Rating: 8/10

The icons were well-implemented with good color choices. Adding context for the icons would further enhance the presentation.

EX-4: Images

Rating: 7.5/10

The image exercise was good, but more information about the changes and additional image properties would make it better.

EX-5: Sample Website

Rating: 7.5/10

The article layout was visually appealing. Including more content, a date, and additional context for images would improve its effectiveness.

EX-6: Design in Practice

Rating: 6/10

The redesign page link was missing, limiting the ability to provide detailed feedback.

Overall Rating: 7/10
