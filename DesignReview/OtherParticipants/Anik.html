Design Review

Anik's Webpage - A Detailed Evaluation

Review Summary

Anik's webpage demonstrates a solid foundation in design principles. However, there is room for improvement in terms of visual impact and usability.

EX-1: Fonts

Rating: 7/10

The font choices are fairly standard, with decent implementation. Improvements in side margins, padding, and line height would enhance the overall design.

EX-2: Colors

Rating: 6/10

The color section suffers from navigation issues, as users need to switch between pages to see differences. The color swatch overwhelms users with too many options at once, lacking clear usage explanations.

EX-3: Icons

Rating: 7/10

The icon solutions were varied but inconsistent. Solution 1's link was broken, while Solution 2 featured an interactive grid with hover effects. Solution 3 showcased a standard navbar with well-designed buttons.

EX-4: Images

Rating: 7.5/10

The image section was well-executed with basic implementations like full-bleed headers and lazy loading. Adding text content alongside images would provide better context.

EX-5: Sample Website

Rating: 7.5/10

The medium clone was aesthetically pleasing, but responsiveness could be improved. Element hierarchy, fonts, and icons were well-implemented overall.

EX-6: Design in Practice

Rating: 6.5/10

The website reviews lacked CSS design, offering little to critique. Redesigns for the IRCTC and faculty pages were well-organized with minimal clutter and clear navigation.

Overall Rating: 7.5/10
