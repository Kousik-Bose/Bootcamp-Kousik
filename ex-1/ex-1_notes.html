HTML Exercise: WEEK 1 Notes

Understanding the Fundamentals of HTML

Semantic HTML Elements

Semantic elements bring clarity and structure to web pages by describing their purpose. Examples include:

  • <header>: Contains introductory or navigational content
  • <nav>: Groups navigation links
  • <main>: Encloses the core content of the document
  • <article>: Represents standalone pieces of content
  • <section>: Defines thematic groups of content
  • <aside>: Used for related or supplementary information
  • <footer>: Holds footer information for a section or the entire page

Header Levels in HTML

HTML offers six levels of headings, from <h1> to <h6>, with <h1> being the most significant and <h6> the least.

Incorporating CSS and JavaScript

Ways to include CSS in HTML:

  • Inline: Adding styles directly using the style attribute
  • Internal: Writing CSS inside <style> tags within the <head>
  • External: Linking a separate CSS file using <link>

Methods to include JavaScript:

  • Inline: Using attributes like onclick
  • Internal: Embedding JavaScript code inside <script> tags
  • External: Linking a JavaScript file using <script src="...">

HTML Validation Techniques

To ensure well-formed HTML, you can:

  • Use tools like the W3C Markup Validation Service
  • Maintain proper nesting of elements
  • Close all open tags
  • Apply valid attributes to elements

HTML5 Template Structure


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>
