Grid Layout & Media Queries - Responsive Design

Grid Layout

Grid layout offers flexibility for creating responsive designs. The property grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); automatically adjusts the columns to fit the screen, ensuring they are at least 200px wide.

Media Queries

Media queries allow styling based on screen size. For smaller screens, use:

@media (max-width: 600px) {
  .container { grid-template-columns: 1fr; }
}

This adjusts the grid layout to a single column on mobile devices.

Challenges

  • Grid items sometimes don't align as expected; using grid-auto-rows helped me here.
  • Testing on real devices is crucial to ensure proper display on all screens, so that was a challenging part for me.
