Understanding CSS Units and Creating Responsive Layouts

Learnings

Level 1

  • height: 100vh; ensures the body takes up the full height of the viewport, adjusting dynamically with screen size.
  • display: flex; is used to enable Flexbox, which makes it easier to center elements.
  • justify-content: center; aligns the content horizontally in the center of the viewport.
  • align-items: center; vertically centers the content within the body.
  • Using percentages for widths allows elements to scale depending on the available space.

Level 2

  • clamp() allows dynamic scaling of properties like font size, adjusting based on viewport width.
  • The min and max functions help define a range for properties, ensuring they don't grow or shrink beyond a specified limit.
  • Viewport-based units like vw are used to scale text sizes relative to the screen width, providing better adaptability.

Level 3

  • CSS Grid is a powerful tool to create responsive, flexible layouts that adjust automatically to different screen sizes.
  • Using grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); allows columns to fit based on the available screen width, adapting to both small and large screens.
  • Media queries provide an additional layer of responsiveness, allowing for styles to be applied based on specific screen dimensions.
  • Fluid typography can be achieved using em and clamp(), which scale text according to the parent element or viewport size, ensuring readability on all devices.
